# reptile

#### 介绍
爬虫项目，微信公众号文章爬虫，网站文章爬虫，群发邮件系统

#### 软件架构
springBoot 单项目架构


#### 安装教程

1.  git下载源码
2.  maven构建
3.  idea-java运行

#### 使用说明

1.  core核心包 java主方法运行
2.  spring boot 主类运行（只加载了微信公众号爬虫的定时任务）

#### 功能简介

简单的爬虫系统和邮件系统
1.爬虫分为微信公众号爬虫和和普通网页爬虫(主要通过selenium和jsoup实现)
2.java邮箱发送系统，可以实现邮件群发(主要通过selenium和jsoup实现)
