神经网络应用于机器翻译

2003 年 2 月，神经网络开始应用于机器翻译。约书亚·本吉奥等发表论文“A Neural Probabilistic Language Model”。他们的研究团队基于神经网络开发出了一个语言模型，改善了统计机器翻译中的问题，这项研究为未来神经网络在机器翻译领域的应用奠定了基础。
